# نظام بَسْمَة AI 🦷
### منصة سحابية متكاملة لإدارة عيادات طب الأسنان

هذا المشروع هو تطبيق ويب متطور صمم خصيصاً لتنظيم العمل السريري في عيادات الأسنان، برؤية وتطوير **الدكتور زكريا محمد عطوني**.

## 🌟 المميزات الرئيسية:
- **سجلات سحابية:** حفظ بيانات المرضى ومزامنتها لحظياً عبر Firebase.
- **مساعد تشخيص ذكي:** يدعم الطبيب في تحليل الأعراض السريرية.
- **إدارة المواعيد والمخزون:** واجهة سهلة لإدارة العمليات اليومية للعيادة.
- **وصول عالمي:** التطبيق يعمل من أي مكان ومن أي جهاز.
